MD Viewer — Privacy Policy

Last updated: February 26, 2026

Overview

MD Viewer is a macOS desktop application for viewing Markdown files with Notion-style rendering. Your privacy is important to us.

Data Collection

MD Viewer does not collect, transmit, or store any personal data. The app operates entirely on your local device.

File Handling

Markdown files you open are read locally and rendered within the app. Files are never uploaded, transmitted, or shared with any external service. MD Viewer has read-only access to files you explicitly open via the file dialog or drag-and-drop.

Network Communication

MD Viewer makes no network connections. All rendering is done locally with bundled libraries. No internet connection is required to use the app.

Analytics and Tracking

MD Viewer contains no analytics, no tracking, no telemetry, and no advertising.

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above.

Contact
